Các bước để chạy chức năng phân loại personal-blog và no-personal-blog
- Đưa danh sách blogs chưa được phân loại vào file "/data/new-users.txt"
- double-click vào file "run.bat" để chạy phân loại.

Output: 
- file "/data/contact-users.txt" chứa những personal-blogs
- file "/data/not-contact-users.txt" chứa những blogs không phải personal

(file "/data/checked-users.txt" là file chứa những blogs trong "/data/new-users.txt" mà đã được phân loại)
